## How to Deploy

### Option 1: GitHub Pages (Recommended)

1. Create a GitHub account at [github.com](https://github.com)
2. Create a new repository named `username.github.io` (replace `username` with your GitHub username)
3. Clone the repository to your computer
4. Copy the `index.html` file into your repo folder
5. Push to GitHub:
   ```bash
   git add index.html
   git commit -m "Add game design portfolio"
   git push origin main
   ```
6. Your site will be live at `https://username.github.io`

### Connect Your Custom Domain

1. Go to your GitHub repository Settings → Pages
2. Under "Custom domain", enter your domain name (e.g., `myportfolio.com`)
3. Update your domain's DNS settings to point to GitHub Pages (GitHub will provide instructions)

### Option 2: Netlify

1. Go to [netlify.com](https://netlify.com)
2. Sign up (free)
3. Drag and drop the `index.html` file
4. Get a live URL instantly
5. Connect your custom domain in Netlify settings

### Option 3: Any Web Host

Upload `index.html` to any web hosting provider and point your domain to it.
